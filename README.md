Профессиональная социальная сеть
Описание:
Сеть для профессионалов: профили, связи, посты, комментарии, уведомления, поиск по навыкам и компаниям.

Сущности:
  Пользователь (профиль: ФИО, навыки, город, описание)
  Связь (друг, коллега, статус)
  Публикация (автор, текст, вложения, уровень видимости)
  Комментарий (пост, автор, текст)
  Реакция (лайк, дизлайк и т. п.)
  Уведомление (пользователь, событие, прочитано/непрочитано)
  Компания, вакансия

  
Функциональность:
  CRUD  для всех сущностей
  Поиск по профилям (полнотекстовый поиск, фильтры по навыкам)
  Формирование ленты новостей (сложные запросы с сортировкой и рейтингами)
  Кэширование ленты и профилей
  Реальное время: обновление уведомлений через WebSocket/Mercure
  Логирование модерации и редактирования контента
  Авторизация: пользователь, модератор, администратор
  Асинхронные задачи: формирование дайджестов, обработка вложений
  Импорт списка пользователей, экспорт профиля (GDPR)
  Тесты: корректность ленты, проверки прав
  CI: автоматическая сборка и тесты
